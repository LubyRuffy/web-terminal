# Web Terminal Demo

> a demo project show how to communicate between xtermjs client and golang websocket server.

* Frontend: Reactjs with Xtermjs
* Backend: Golang with beego framework

## Screenshot

![](img/demo.png)

## How to start

**Frontend**

```shell script
cd frontend
npm start
```

**Backend**

```shell script
cd backend
go run main.go
```
